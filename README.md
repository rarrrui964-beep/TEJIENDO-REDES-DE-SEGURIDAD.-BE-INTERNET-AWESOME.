# TEJIENDO-REDES-DE-SEGURIDAD.-BE-INTERNET-AWESOME.
Comprender la importancia de la seguridad y la privacidad en la red. Fomentar la amabilidad, el respeto y la netiqueta en entornos digitales. Identificar noticias falsas y posibles fraudes online (phishing). Trabajar en equipo para resolver retos gamificados. Expresar conocimientos mediante herramientas creativas digitales.
